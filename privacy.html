プライバシーポリシー

最終更新日: 2026年4月24日

1. 取得する情報

当サービスでは、新規登録時に以下の情報を取得します。

  • メールアドレス（必須・ログインID および通知用）
  • パスワード（必須・Supabase Auth により暗号化ハッシュ化して保存。平文では保持しません）
  • 氏名またはニックネーム（必須）
  • 所属大学名（必須）
  • 卒業予定年度（必須）
  • 第一志望の業界（任意）
  • テスト受験結果（正答率・所要時間など）
  • IP アドレス、User Agent、アクセス日時（安全管理および同意記録のため）

2. 利用目的

  • 本サービスの提供および成績分析のため
  • 利用者の同意に基づき、提携企業に対して匿名化されたプロフィール情報（大学名、卒業年度、テスト成績等）を公開し、スカウトメッセージを受信するため
  • サービス改善・統計分析のため（個人を特定できない形での集計処理）
  • 不正利用の防止および安全管理のため
  • 法令等に基づく対応のため

3. 第三者への提供（スカウト機能）

スカウト機能の利用に同意された利用者に限り、提携企業に対して以下の情報を開示します。 氏名・メールアドレスなどの直接連絡先は、利用者がスカウトメッセージに返信する選択をするまで公開されません。

  • 匿名化されたニックネーム
  • 所属大学名
  • 卒業予定年度
  • 第一志望の業界（任意回答）
  • テスト成績のサマリ

スカウト機能の利用同意は、マイページからいつでも取り消すことができます。

4. 安全管理措置

  • パスワードは Supabase Auth により一方向ハッシュで保存し、運営者もアクセスできません
  • データベースアクセスは Supabase の Row Level Security により、各ユーザーは自身のデータ以外を閲覧不可
  • 通信は HTTPS で暗号化
  • 管理画面は管理者パスワード認証・セッショントークン・アクセス元 IP レートリミット等で多層防御

5. 利用者の権利（開示・訂正・削除）

個人情報保護法に基づき、利用者は自身の情報について開示・訂正・削除の請求が行えます。 マイページから退会処理を行うと、本人を特定できる情報は原則として削除されます （バックアップ・法令上の保存義務がある場合を除く）。

6. お問い合わせ

本ポリシーに関する質問・請求は、運営者までメールでお問い合わせください。

7. 改定

本ポリシーは、法令の変更やサービス改善に伴い改定される場合があります。 重要な変更を行う際には、本ページにて通知します。
